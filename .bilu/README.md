# Docs

This folder documents the `bilu` project and its CLI/board UX.

## Contents
- `.bilu/README.md`: Overview of Bilu, features, installation, usage, and configuration.
- `.bilu/storage/prd.md`: Product requirements document for Bilu.
- `.bilu/board/`: Board files defining tasks for Bilu to work on.
