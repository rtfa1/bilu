# Docs

This folder documents the `bilu` project and its CLI/board UX.

## Contents

- `src/docs/bilu-cli.md`: CLI usage and installation notes.
- `src/docs/phases/00-board-ui-overview.md`: Overall plan for the shell-only board UI.
- `src/docs/phases/01-data-contract.md`: Board data model, normalization rules, and precedence.
- `src/docs/phases/02-cli-and-modules.md`: Shell module layout for `bilu board` and argument parsing.
- `src/docs/phases/03-rendering-table-and-kanban.md`: Non-interactive renderers (table + kanban).
- `src/docs/phases/04-interactive-tui.md`: Interactive keyboard-driven TUI design (shell only).
- `src/docs/phases/05-persistence-and-editing.md`: How edits are persisted safely to disk.
- `src/docs/phases/06-testing-and-docs.md`: Test strategy and documentation checklist.
